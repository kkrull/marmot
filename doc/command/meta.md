# `marmot meta`

`marmot meta` - Information about the meta repo

- [`marmot meta home`](./meta-home.md)

## SYNOPSIS

```sh
marmot meta --help
marmot meta sub-command [options...]
```

## SUB-COMMANDS

```text
home          Show the base directory of the Meta Repo
```

## OPTIONS

```text
--help        Show help
```
