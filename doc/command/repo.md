# `marmot repo`

`marmot repo` - Work with repositories

- [`marmot repo list`](./repo-list.md)
- [`marmot repo register`](./repo-register.md)

## SYNOPSIS

```sh
marmot repo --help
marmot repo sub-command [options...]
```

## SUB-COMMANDS

```text
list          List repositories
register      Register repositories to manage
```

## OPTIONS

```text
--help        Show help
```
