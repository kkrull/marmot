# NAME

**marmot meta home** - Show path to Meta Repo

# SYNOPSIS

**marmot meta home** \[**\--help**\]\
**marmot meta home**

# DESCRIPTION

**marmot meta home** prints the base directory of the Meta Repo.

# OPTIONS

**\--help**  
Show help
